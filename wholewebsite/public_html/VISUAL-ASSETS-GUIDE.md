## 💡 Pro Tips

1. **Consistency is Key**
   - Use same photographer/style
   - Maintain color grading
   - Match lighting across images

2. **Optimization**
   - Always compress before upload
   - Use WebP with JPG fallback
   - Implement lazy loading

3. **Accessibility**
   - Add alt text to all images
   - Ensure sufficient contrast
   - Provide text alternatives

4. **Loading Performance**
   - Use CDN for assets
   - Implement progressive loading
   - Optimize for mobile first

5. **Copyright**
   - Always verify license
   - Keep attribution if required
   - Document source of each image
