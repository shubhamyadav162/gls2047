## 📐 Image Specifications

### General Guidelines
- **Formats:** Use PNG for logos/graphics, JPG for photos
- **Optimization:** Compress all images for web (use tools like TinyPNG, ImageOptim)
- **Color Space:** sRGB
- **File Naming:** Use lowercase with hyphens (e.g., `paniit-logo.png`)
- **Transparent Backgrounds:** Required for all logos

### Recommended Tools
- **Image Compression:** TinyPNG, Squoosh, ImageOptim
- **Image Editing:** Photoshop, GIMP, Canva
- **Format Conversion:** CloudConvert, XnConvert
- **Favicon Generation:** RealFaviconGenerator

## 🎨 Brand Asset Guidelines

When creating or selecting images, ensure they:
1. Align with brand colors (Royal Blue #0D47A1, Gold #D4AF37)
2. Are high quality and professional
3. Have proper licensing/permissions
4. Are optimized for web performance
5. Work well with gradient overlays (for hero section)
